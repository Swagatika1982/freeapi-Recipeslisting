Meals & Recipes Listing App
A simple and responsive React application that fetches and displays meals/recipes using the FreeAPI Meals API.
This project demonstrates:


Fetch API usage in React


Dynamic data rendering


Loading & error handling


Responsive card layouts


Clean UI design



Features


Fetch meals from API


Display meal image, title, category, and origin


Show recipe instructions preview


Responsive grid layout


Refresh meals button


Loading and error states



Tech Stack


React


Vite


JavaScript


CSS



API Used
Meals API:
https://api.freeapi.app/api/v1/public/meals

Installation
Clone the repository:
git clone https://github.com/Swagatika1982/freeapi-Recipeslisting.git
Go to project folder:
cd meals-listing-app
Install dependencies:
npm install
Start development server:
npm run dev
